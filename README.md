dialog types
============

Flow types for [dialog](https://dlg.im) projects.

Installation
------------

```
npm install --save @dlghq/dialog-types
```

Usage
-----

```js
/*
 * @flow
 */

import type { User } from '@dlghq/dialog-types';

const user: User = getUser();
```
